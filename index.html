Seal Master Instruction Note

Korea pc방 မှာ ယူတဲ့အခါ ပုံမှန်တွေထက်လိုင်းကောင်းတာဖြစ်လို့ 4:15 မှ ဝင်ရင် အဆင်ပြေပါပြီ။ (စောဝင်တဲ့အခါ refresh လုပ်ရာကနေ server က WiFi ကို bot အမှတ်နဲ့ ban သွားတာမျိုးတွေ ခဏခဏ ကြုံတွေ့ရတတ်ပါတယ်)

မြန်မာမှာ သို့မဟုတ် ကိုယ့်အိမ် WiFi နဲ့ ဖုန်း/ pc စတာတွေနဲ့ ယူမယ်ဆိုရင်တော့ အနည်းဆုံး နာရီဝက် ကြိုဝင်ထားတာ အကောင်းဆုံးပါ။ code မကျတဲ့ပြသနာကို မပူရတော့ပါဘူး။

အရေးကြီးဆုံးက သံရုံးဖွင့်ပေးတဲ့အချိန်မှာ မြန်မာသံရုံးချက်ချင်းပေါ်ဖို့နဲ့ 404 error မတက်ဖို့က နံပါတ် ၁ ပါ။ အဲ့တာကြောင့် ကြိုပြီး log in ဝင်ထားပြီးရင် ခဏခဏ refresh လုပ်ခြင်းကို ရှောင်ရပါမယ်။ မလုပ်ဘဲ ပစ်ထားတာမျိုးလည်းမဟုတ်ပါဘူး။ 1:50 or 4:20 မတိုင်ခင်ထိ ၅ မိနစ်တစ်ခါ ၇ မိနစ်တစ်ခါမှ လုပ်ပေးပြီး screen မပိတ်အောင် page မကျသွားအောင် တော်ရုံလောက် လုံပေးရင် ရပါတယ်။

1:50 or 4:20 ပြီးရင်တော့ လိုင်းက ပိုကြပ်လာပြီမလို့ ၂ မိနစ်ခြား သို့ ၃ မိနစ်ခြား လုပ်ပေးပြီး 1:59 / 4:29 မှာ Make a reservation page မှာစောင့်ပြီး အချိန်စေ့တာနဲ့ ဝင်ကြည့်လိုက်ပါ။ အဲ့မှာ အကုန်ပြုံဝင်ကြတာဖြစ်လို့ လိုင်းကြပ်ပြီး အဝိုင်းလည်နေနိုင်ပါတယ်။

စိတ်လောပြီး လိုက်နှိပ်တာထက် အဝိုင်းလည်နေတာကို စောင့်ပေးပါ။ log in နေရာ embassy ရွေးတဲ့နေရာ ရောက်ပြီဆိုရင် ချက်ချင်း နိုင်ငံရွေးလိုက်ပါ။

အကယ်၍ သံရုံးက ကွက်တိမပေါ်ခဲ့ရင် Ctrl + R နှင့် Make a reservation page နေရာကို refresh လုပ်တာကို အလှည့်စီလုပ်ပြီး နိုင်ငံရွေးကြည့်ပါ။ အဓိကအချက်က ဆက်တိုက်နှိပ်တာမဟုတ်ဘဲ အဝိုင်းလည်တာပြီးတဲ့ထိစောင့်ပြီး 2 seconds လောက်နားလုပ်တာ ကောင်းပါတယ်။
