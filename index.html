မြန်မာမှာ သို့မဟုတ် ကိုယ့်အိမ် WiFi နဲ့ ဖုန်း/ pc စတာတွေနဲ့ ယူမယ်ဆိုရင်တော့ အနည်းဆုံး နာရီဝက် ကြိုဝင်ထားတာ အကောင်းဆုံးပါ။ code မကျတဲ့ပြသနာကို မပူရတော့ပါဘူး။

အရေးကြီးဆုံးက သံရုံးဖွင့်ပေးတဲ့အချိန်မှာ မြန်မာသံရုံးချက်ချင်းပေါ်ဖို့နဲ့ 404 error မတက်ဖို့က နံပါတ် ၁ ပါ။ အဲ့တာကြောင့် ကြိုပြီး log in ဝင်ထားပြီးရင် ခဏခဏ refresh လုပ်ခြင်းကို ရှောင်ရပါမယ်။ မလုပ်ဘဲ ပစ်ထားတာမျိုးလည်းမဟုတ်ပါဘူး။ 1:50 or 4:20 မတိုင်ခင်ထိ ၅ မိနစ်တစ်ခါ ၇ မိနစ်တစ်ခါမှ လုပ်ပေးပြီး screen မပိတ်အောင် page မကျသွားအောင် တော်ရုံလောက် လုံပေးရင် ရပါတယ်။
